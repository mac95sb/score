# __NAME__

A static Score site.

## Project structure

- `Sources/Views/Pages/` — Page views
- `Sources/Views/Components/` — Reusable components
- `Sources/Application.swift` — App entry point
- `Content/posts/` — Markdown content
- `Public/` — Static assets

## Running locally

```bash
score dev
```

## Building for production

```bash
score build
```
